# Data Directory

This directory should contain the historical solar energy production data files.

## Expected Files:
- `PV Plants Datasets.xlsx` - Historical PV production data
- `PV Plants Metadata.xlsx` - PV installation metadata
- Any additional CSV or Excel files with energy production data

## Note:
Data files are excluded from git tracking due to their large size and potentially sensitive nature.
Please ensure you have the necessary data files in this directory before running the application.
