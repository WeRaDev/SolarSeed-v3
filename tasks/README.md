# Tasks
This directory stores sprint-ready and backlog work items for `SolarSeed-v3`.

## Lifecycle
`Backlog -> Ready -> In Progress -> Review -> Validated -> Done`

## Required fields per task
- Problem statement
- Scope and non-scope
- Acceptance criteria
- Dependencies
- Validation plan

## Notes
- Keep tasks tied to TRL branch context (`trl4` or `trl5`) when applicable.
- Link runbook and rollback references for operational changes.
